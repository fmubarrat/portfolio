Neuro-Motor Assessment · Sketch Process Analysis NEW 2025

From Drawing Product to Drawing Process: Parkinson's Motor Assessment

Two-part feasibility study combining 2D sketch product analysis (204 real PD/healthy drawings) with novel 3D video-based hand kinematic process analysis using MediaPipe. Demonstrated that micrographia is detectable from hand movement alone (p=0.019, Cohen's d=3.2) — without analyzing written text. Tremor features showed >1000% magnitude increases under simulated conditions. Requires only a smartphone camera ($0 hardware).
